# VMC Helty Flow Brand

Questa cartella contiene le risorse del brand per il progetto VMC Helty Flow.

## Icona
L'icona dovrebbe essere un file PNG di 256x256 pixel che rappresenta il sistema di ventilazione meccanica controllata Helty Flow.

Per aggiungere un'icona personalizzata:
1. Crea un file PNG chiamato `icon.png` in questa cartella
2. L'icona dovrebbe essere 256x256 pixel
3. Usa colori che rappresentino la ventilazione (blu, azzurro, grigio)
